# 김민수 - 자기소개서

소프트웨어 개발자 김민수의 포트폴리오 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)

## 로컬에서 실행하기

1. 저장소 클론:
```bash
git clone https://github.com/[사용자명]/[저장소명].git
```

2. 브라우저에서 `index.html` 파일 열기

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소 생성
2. 이 폴더의 모든 파일을 저장소에 업로드
3. 저장소 Settings → Pages
4. Source를 `main` 브랜치로 설정
5. 저장 후 몇 분 뒤 `https://[사용자명].github.io/[저장소명]`에서 확인
